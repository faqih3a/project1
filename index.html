Sebuah aplikasi berbasis WEB yang berfungsi untuk mempermudah pasien untuk mengakses puskesmas secara online

  • Advantages

    Memudahkan pasien untuk mengakses data puskesmas secara online

F.A.Q

Frequently Asked Questions

Beberapa pertanyaan tentang Aplikasi ini

  • Apa fungsi dari E Puskesmas ini?

    Salah satu fungsinya adalah menampung pendataan yang ada di puskesmas,posyandu,dan juga pustu secara cloud computing sehingga data tidak akan hilang selama tidak ada masalah pada server cloud

  • Tujuan dari aplikasi ini apa?

    Mempermudah mencari dan melihat data-data pasien yang telah di upload pada website ini

  • Siapa saja yang dapat mengakses website ini?

    Ada 3 tingkatan; 1. Super Admin yang dapat mengakses seluruh data pengguna admin, 2. Admin yang menginput data pasien, 3. Member yaitu pasien sendiri

  • Alasan membuat website aplikasi ini apa?

    Tugas kuliah dari mata kuliah Pemrograman Web Dasar
